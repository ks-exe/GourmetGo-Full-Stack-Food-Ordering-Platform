Smart Food Recommendation System

This module uses **Content-Based Filtering** with `CountVectorizer` and `cosine_similarity` to calculate resemblance scores of food items using their Name, Category, and Description. Recommendations are dynamically boosted by orders count (popularity) loaded from the database.

GET /api/ml/recommendations?query=<food_name>
Retrieves top 5 similar items to the requested food item, sorted by a weighted content similarity + popularity score.
// Sample Response [ { "id": 3, "name": "Zinger Stack", "category": "Fast Food", "price": 950, "image": "...", "description": "..." }, { "id": 1, "name": "Classic Cheeseburger", "category": "Fast Food", "price": 850, "image": "...", "description": "..." } ]
GET /api/ml/trending
Retrieves top 6 most ordered items (trending foods) in descending order of orders from the MongoDB database.

Best Selling Food Prediction

This module uses a **RandomForestRegressor** model trained on a 120-day historical sales dataset. It evaluates the impact of weather conditions, weekends, occasions, and previous sales to forecast sales demand, categorize demand levels, and generate stock alerts.

POST /api/ml/predict-sales
Returns a detailed demand forecast for all dishes in the database.
// Request Payload { "weather": "Hot", "temperature": 35, "weekend": "Yes", "occasion": "Eid" } // Sample Response { "weather": "Hot", "temperature": 35, "weekend": "Yes", "occasion": "Eid", "predictions": [ { "id": 21, "name": "Iced Caramel Macchiato", "category": "Drinks", "expected_sales": 48, "demand_level": "High Demand", "stock_recommendation": "Increase stock by 50% (High Demand alert)", "alert": true } ] }
